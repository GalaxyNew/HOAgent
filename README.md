# HOAgent

HOAgent 是查理资讯的 Agent 运营控制台项目入口。

当前仓库处于 **M0 治理基线**：仅登记项目边界、进度与协作规范；不包含产品实现代码。

## 开发边界

- 阶段 1 仅实现可重建、默认脱敏的只读投影。
- 权威源保持不变；投影数据可删除并从授权来源重建。
- 禁止提交凭据、Cookie、Token、私人记忆或交易正文。
