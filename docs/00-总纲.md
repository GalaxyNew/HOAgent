# HOAgent 项目总纲

## 定位

HOAgent 是查理资讯的运营控制台项目。首期目标是建立**只读、可重建、默认脱敏**的数据投影与查询能力，服务任务、Agent/组织、运行时元数据及授权业务双轴数据的查看。

## 不变边界

1. 任务卡、Agent Registry、Vault、运行时及带 `source_ref` 的授权业务来源仍为权威源。
2. 投影库不是事实源，可随时删除后由授权来源重建。
3. 首期不提供写业务端点、不开放公网、不引入多节点或向量库。
4. 不读取、保存或索引 Token、Cookie、密码、私人记忆正文、未授权会话正文、订单或交易正文。
5. 所有代码变更必须关联 GitHub Issue、走分支与 PR，并保留可验证回滚路径。
